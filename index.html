Features

Built for engineers who think in prompts.

Every feature is designed to make your code work better with large language models — no fluff, no bloat.

Instant conversion

Flatten any repo in under 2 seconds. We stream files in parallel, dedupe paths, and assemble output on the fly.

Private by design

Your code never touches our servers. Everything runs in your browser via the GitHub API with your personal token.

Smart filtering

Glob-style exclude patterns, size limits, and automatic binary detection. Keep only what matters for your prompt.

Token estimator

Know exactly how many tokens your repo will cost before you paste it. Accurate GPT-4 and Claude tokenizers built in.

Three output formats

Markdown with code fences, XML tags for Claude, or plain text. Pick the format your model understands best.

Branch & commit aware

Pin to a specific branch, tag, or commit SHA. Compare two versions side-by-side or extract exactly the state you need.

Keyboard-first

Cmd+K to convert, Cmd+C to copy, Cmd+S to download. Power users never reach for the mouse.

API access

Programmatic endpoints for CI pipelines, Slack bots, and internal tools. 50k conversions per month on Pro.

Conversion history

Your last 100 conversions are stored locally and synced across devices. Re-download anytime, no re-conversion needed.

How it works

Three steps. Ten seconds.

01

Paste your repo URL

Drop in any public GitHub URL — or use a personal access token for private repositories. Supports monorepos and subdirectories.

02

Tune the options

Pick a format, set exclude patterns, cap file sizes. We'll show you a live token estimate as you tweak.

03

Copy or download

Grab the entire output to your clipboard, download as .txt, or hit the API for automation.
